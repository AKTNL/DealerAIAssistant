# 模型设置实施计划

> **面向代理工作器：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐步实施此计划。步骤使用复选框（`- [ ]`）语法进行追踪。

**目标：** 添加浏览器本地模型设置，包含顶栏设置面板、立即下一条消息激活，以及在聊天可用前的专用连接测试。

**架构：** 将浏览器作为已保存配置的事实来源，以 `localStorage` 为持久化存储，以响应式内存副本为活跃会话。随每次聊天请求发送已保存的配置，并将后端的模型创建/测试移入请求驱动服务，以便在主流程中没有系统默认模型路径。

**技术栈：** Vue 3、Vitest、Vue Test Utils、Spring Boot 3.4、Spring Validation、Spring AI 1.0、Maven、H2。

```js
export function isModelSettingsComplete(settings) {
  return Boolean(settings?.baseUrl && settings?.apiKey && settings?.model);
}

export function readModelSettings() {
  const raw = window.localStorage.getItem(STORAGE_KEYS.modelSettings);
  if (!raw) {
    return null;
  }

  try {
    return JSON.parse(raw);
  } catch {
    return null;
  }
}
```
