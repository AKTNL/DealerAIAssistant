# 可折叠侧边栏题库实施计划

> **面向代理工作器：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐步实施此计划。步骤使用复选框（`- [ ]`）语法进行追踪。

**目标：** 将聊天侧边栏转换为六模块可折叠题库，每个模块包含两个本地化问题，被点击的问题有内联等待反馈，并且语言切换时保留展开的模块。

**架构：** 保持当前数据流不变：本地化模块数据保留在 `sidebarFlows.js` 中，`messages.js` 继续将其暴露为 `promptModules`，`ChatView` 继续通过 `submitPrompt(question)` 处理直接提交。将新行为放入 `ExampleSidebar.vue`，通过按稳定展开模块标识和瞬态等待问题标识在本地追踪状态，使语言变化可以替换可见文本而不重置 UI 状态。

**技术栈：** Vue 3 `<script setup>`、Vitest、Vue Test Utils、Vite CSS（在 `frontend/src/style.css` 中）

- [ ] **步骤 3：编写最小手风琴实现**

重构 `frontend/src/components/layout/ExampleSidebar.vue`，使其在本地响应式状态中管理展开模块索引和等待问题标识：

```vue
<script setup>
import { ref, watch } from "vue";

const props = defineProps({
  activeSessionLabel: { type: String, required: true },
  dictionary: { type: Object, required: true },
  isSending: { type: Boolean, default: false },
  showMobileSidebar: { type: Boolean, default: false }
});

const emit = defineEmits(["close", "new-chat", "submit-prompt", "toggle-sidebar"]);

const expandedModuleIndex = ref(null);
const pendingQuestionKey = ref("");

function toggleModule(index) {
  expandedModuleIndex.value = expandedModuleIndex.value === index ? null : index;
}

function buildQuestionKey(moduleIndex, questionIndex) {
  return `${moduleIndex}:${questionIndex}`;
}

function handleQuestionClick(question, moduleIndex, questionIndex) {
  pendingQuestionKey.value = buildQuestionKey(moduleIndex, questionIndex);
  emit("submit-prompt", question);
}

watch(
  () => props.isSending,
  (value) => {
    if (!value) {
      pendingQuestionKey.value = "";
    }
  }
);
</script>
```

按索引渲染模块，使语言变化保留打开的模块：

```vue
<section v-if="dictionary.promptModules?.length" class="sidebar-section">
  <div class="section-head">
    <span>{{ dictionary.sidebarSection }}</span>
  </div>

  <div
    v-for="(module, moduleIndex) in dictionary.promptModules"
    :key="moduleIndex"
    class="sidebar-module-card"
  >
    <button
      class="sidebar-module-toggle"
      type="button"
      :aria-expanded="String(expandedModuleIndex === moduleIndex)"
      @click="toggleModule(moduleIndex)"
    >
      <span>{{ module.title }}</span>
      <span
        :class="['material-icons', 'sidebar-module-chevron', { open: expandedModuleIndex === moduleIndex }]"
      >
        expand_more
      </span>
    </button>

    <div v-if="expandedModuleIndex === moduleIndex" class="sidebar-module-questions">
      <button
        v-for="(question, questionIndex) in module.questions"
        :key="questionIndex"
        :class="[
          'prompt-card',
          'sidebar-module-question',
          { 'is-pending': pendingQuestionKey === `${moduleIndex}:${questionIndex}` }
        ]"
        type="button"
        :disabled="isSending"
        @click="handleQuestionClick(question, moduleIndex, questionIndex)"
      >
        <span>{{ question }}</span>
        <span
          v-if="pendingQuestionKey === `${moduleIndex}:${questionIndex}`"
          class="sidebar-question-spinner"
          aria-hidden="true"
        ></span>
      </button>
    </div>
  </div>
</section>
```
