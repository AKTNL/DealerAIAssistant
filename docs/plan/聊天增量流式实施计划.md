# 聊天增量流式实施计划

> **面向代理工作器：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐步实施此计划。步骤使用复选框（`- [ ]`）语法进行追踪。

**目标：** 将 `/api/chat/stream` 变为真正的增量助手正文流式输出，在失败时保持部分内容可见，尊重滚动锁定，并强制执行硬输出限制。

**架构：** 保持现有 SSE 封装和当前 `progress` / `thinking` / `message` / `done` / `error` 事件名称。将后端从阻塞式完整回复生成迁移到 `ChatModel.stream(...)` 片段转发，仅累积最终正文用于成功持久化，并教会前端追加流式正文片段，同时仅在用户已锚定在底部时自动滚动。

**技术栈：** Spring Boot 3.4、Spring AI 1.0.0、Vue 3、Vitest、JUnit 5、Mockito

```java
static final int MAX_STREAMED_REPLY_CHARS = 32_000;

private String extractChunkText(ChatResponse response) {
    if (response == null || response.getResult() == null || response.getResult().getOutput() == null) {
        return "";
    }
    return response.getResult().getOutput().getText();
}

private void appendChunk(StringBuilder accumulator, String chunk) {
    if (accumulator.length() + chunk.length() > MAX_STREAMED_REPLY_CHARS) {
        throw new IllegalStateException("流式回复超出允许的输出限制。");
    }
    accumulator.append(chunk);
}
```

- [ ] **步骤 2：用 `chatModel.stream(...)` 替换已配置的流式路径**

```java
private void streamConfiguredReply(
        BufferedWriter writer,
        ChatRequest request,
        String language,
        boolean analyticsRequested,
        AnalyticsPlan analyticsPlan
) throws IOException {
    ChatModel chatModel = modelConfigService.createChatModel(request);
    String sessionHistory = formatSessionHistory(request.sessionId(), language);
    String userPrompt = analyticsRequested
            ? promptFactory.buildGroundedModelPrompt(language, request.message(), sessionHistory,
                    analyticsPlan.groundedReference())
            : promptFactory.buildConversationModelPrompt(language, request.message(), sessionHistory);

    StringBuilder accumulatedReply = new StringBuilder();
    Prompt prompt = new Prompt(List.of(
            new SystemMessage(promptFactory.buildSystemPrompt(language)),
            new UserMessage(userPrompt)
    ));

    for (ChatResponse chunkResponse : chatModel.stream(prompt).toIterable()) {
        String chunk = extractChunkText(chunkResponse);
        if (chunk == null || chunk.isBlank()) {
            continue;
        }
        appendChunk(accumulatedReply, chunk);
        writeEvent(writer, "message", chunk);
    }

    String finalReply = ensureFollowUpQuestions(accumulatedReply.toString(), language, analyticsRequested);
    String streamedTail = finalReply.substring(accumulatedReply.length());
    if (!streamedTail.isEmpty()) {
        writeEvent(writer, "message", streamedTail);
        accumulatedReply.append(streamedTail);
    }
    writeEvent(writer, "done", "[DONE]");
    sessionMemoryService.addAssistantMessage(request.sessionId(), accumulatedReply.toString());
}
```

- [ ] **步骤 3：保持降级流程单次化，使写入器可扩展用于未来思考片段**

```java
private void writeChunkedEvent(BufferedWriter writer, String event, String chunk) throws IOException {
    if (chunk == null || chunk.isBlank()) {
        return;
    }
    writeEvent(writer, event, chunk);
}

public void streamChat(ChatRequest request, OutputStream outputStream) throws IOException {
    String language = languageDetector.detectLanguage(request.message());
    boolean analyticsRequested = looksLikeAnalyticsRequest(request.message());
    AnalyticsPlan analyticsPlan = analyticsRequested ? analyticsService.plan(request.message(), language) : null;
    sessionMemoryService.addUserMessage(request.sessionId(), request.message());

    try (BufferedWriter writer = new BufferedWriter(new OutputStreamWriter(outputStream, StandardCharsets.UTF_8))) {
        List<String> progressMessages = buildModelProgressMessages(language, analyticsRequested);
        writeEvent(writer, "progress", progressMessages.getFirst());

        if (!hasCompletedModelSettings(request)) {
            GeneratedReply generatedReply = analyticsPlan != null
                    ? new GeneratedReply(analyticsPlan.fallbackReply().trim(), analyticsPlan.progressMessages(), analyticsPlan.visibleThinking())
                    : generateReply(request, language, false);
            writeChunkedEvent(writer, "message", "<think>" + generatedReply.visibleThinking() + "</think>");
            writeChunkedEvent(writer, "message", generatedReply.reply());
            writeEvent(writer, "done", "[DONE]");
            sessionMemoryService.addAssistantMessage(request.sessionId(), generatedReply.reply());
            return;
        }

        writeChunkedEvent(writer, "message", "<think>" + buildModelVisibleThinking(language, analyticsRequested) + "</think>");
        streamConfiguredReply(writer, request, language, analyticsRequested, analyticsPlan);
    } catch (Exception exception) {
        writeEvent(writer, "error", describeStreamFailure(exception));
    }
}
```

```javascript
const SCROLL_LOCK_THRESHOLD = 40;

const hasUnreadMessages = ref(false);
const isPinnedToBottom = ref(true);

function isViewportPinned() {
  if (!scrollContainer.value) {
    return true;
  }

  const { scrollHeight, scrollTop, clientHeight } = scrollContainer.value;
  return scrollHeight - (scrollTop + clientHeight) <= SCROLL_LOCK_THRESHOLD;
}

function handleScroll() {
  isPinnedToBottom.value = isViewportPinned();
  if (isPinnedToBottom.value) {
    hasUnreadMessages.value = false;
  }
}

async function jumpToLatest() {
  hasUnreadMessages.value = false;
  isPinnedToBottom.value = true;
  await scrollToBottom();
}
```

- [ ] **步骤 2：追加流式正文片段并尊重滚动锁定**

```javascript
function appendAssistantChunk(message, chunk) {
  message.content += chunk;
  const normalized = normalizeAssistantPayload(message.content);
  message.content = normalized.content;
  message.followUps = normalized.followUps;
  message.html = renderMarkdownLite(normalized.content);
  message.status = "";
}

function maybeScrollAfterChunk() {
  if (isPinnedToBottom.value) {
    scrollToBottom();
    return;
  }
  hasUnreadMessages.value = true;
}

if (event === "message") {
  if (eventText.startsWith("<think>") && eventText.endsWith("</think>")) {
    const thinking = eventText.slice(7, -8).trim();
    assistantMessage.thinking = thinking;
    assistantMessage.thinkingHtml = renderMarkdownLite(thinking);
    maybeScrollAfterChunk();
    return;
  }

  appendAssistantChunk(assistantMessage, eventText);
  maybeScrollAfterChunk();
  return;
}
```
