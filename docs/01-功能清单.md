# 功能清单 — 某豪华汽车品牌经销商 AI 分析助手 (Agent POC)

> 来源：Agent-POC-需求文档 v2026-04-29  
> 说明：本文档梳理 POC 范围内所有功能点，按模块分类。

---

## 1. 登录与鉴权

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F01 | 访问口令验证 | 用户输入口令，POST `/api/auth/verify` 校验，通过后进入系统 | P0 |
| F02 | 会话级登录态 | 登录标记写入 sessionStorage，关闭浏览器即失效 | P0 |
| F03 | 口令错误处理 | 校验失败清空输入框、显示错误提示、输入框抖动动画 | P1 |
| F04 | 防重复提交 | 认证过程中拦截重复提交；网络异常与口令错误统一按失败处理 | P1 |
| F05 | 回车提交 | 口令输入框支持回车键提交 | P2 |
| F06 | API Key 过滤 | 除白名单路径外，其他接口校验 `X-API-Key` 请求头 | P0 |
| F07 | CORS 配置 | 允许 `localhost:5173` / `127.0.0.1:5173` 跨域访问 | P0 |

---

## 2. 聊天对话

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F10 | 自然语言提问 | 用户输入问题，发送到后端 AI 链路 | P0 |
| F11 | SSE 流式回复 | 后端通过 `text/event-stream` 流式返回 AI 回复 | P0 |
| F12 | 流式打字效果 | 前端实时追加展示流式内容，打字光标动画 | P1 |
| F13 | 流式期间输入禁用 | 当存在流式消息时，输入框禁用 | P2 |
| F14 | 多轮对话 | 后端按 sessionId 维持会话记忆，支持上下文连续对话 | P0 |
| F15 | 清空会话 | 前端调用 `DELETE /api/chat/{sessionId}` 清理后端会话，本地重置 | P1 |
| F16 | 清空后提示 | 清空后聊天区显示"会话已清空"提示，生成新 sessionId | P2 |
| F17 | 欢迎语 | 首次进入会话展示默认欢迎语 | P2 |

## 3. AI 回复展示

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F20 | Markdown 渲染 | assistant 消息支持标题、表格、列表、引用、代码块渲染 | P0 |
| F21 | 代码语法高亮 | 代码块使用 highlight.js 高亮显示 | P1 |
| F22 | 思考过程折叠 | `<think>...</think>` 内容默认折叠，点击展开 | P1 |
| F23 | 思考中状态 | 流式期间若思考步骤已返回但正文未到达，展示"思考中" | P2 |
| F24 | 追问按钮 | 从回复中提取 `FOLLOW_UP_QUESTIONS` 区块渲染为可点击按钮 | P0 |
| F25 | 追问兼容 | 支持中英文 heading（"跟进问题""Follow-up Questions"等），数字/无序列表 | P1 |
| F26 | 追问触发 | 点击追问按钮，将问题作为新消息发送 | P0 |

## 8. 配置管理

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F70 | 服务端口配置 | 后端默认 8081，环境变量 `SERVER_PORT` 可覆盖 | P0 |

| F73 | OpenAI 模型配置 | api-key / base-url / 模型名称均可配置 | P0 |
| F74 | Excel 路径配置 | `app.excel.path`，环境变量 `APP_EXCEL_PATH` 可覆盖 | P2 |
| F75 | 模型配置热更新 | 前端模型设置面板可动态修改模型连接参数（加分项） | P2 |

## 9. 非功能需求

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| F80 | 一键启动 | 前后端可本地一键启动 | P0 |

## POC 范围外 (明确不做)

- 用户体系、RBAC、SSO
- 真实 CRM / DMS / CDP 系统对接
- 数据权限隔离
- 持久化聊天记录
- 生产级安全治理
- 复杂监控、审计、告警
- 移动端 App
- 多链路标签页切换（仅需一条 AI 链路）
