📡 系统调度与会话控制 API (Core Control)

定义前端 Web 界面与 Agent Core 之间的通信契约，涵盖鉴权、流式会话管理及状态重置。

POST /api/auth/verify

访问密钥验证

校验用户输入的访问口令，通过后建立 Session 会话。失败触发 401 状态码及前端抖动动效。

POST /api/chat/stream

流式 AI 对话 (SSE)

核心交互接口。基于 text/event-stream 协议实时推送思考进度、正文分片及 [DONE] 标记。

POST /api/chat

同步 AI 对话

标准阻塞式 HTTP 请求。用于简单的问答测试，一次性返回完整消息体，不支持打字机效果。

DELETE /api/chat/{id}

清除会话记忆

物理清除后端内存中的会话上下文。前端点击“清空会话”时调用，同时重置本地 sessionId。

📊 执行与输出规范 (Action Specs)

定义 Agent 在执行阶段的数据加工公式及回复范式约束。

f(x) 核心 KPI 算法引擎

🎯 目标达成率
实际赢单数 ÷ 下达目标量 × 100%
🔽 漏斗流失率
(当前阶段数 - 下一阶段数) ÷ 当前阶段数 × 100%
⚠️ 任务超期率
Overdue任务数 ÷ 该维度总任务数 × 100%
📢 活动产出比 (ROI)
实际产出商机 ÷ 规划新客目标 × 100%
📈 线索转化率
IsConverted=True 数量 ÷ 总线索量 × 100%
